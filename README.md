Project Description
  Blueprint is a project management tool designed for students to manage their projects and groups. It allows users to make and assign tasks for project development.
  Tasks that are assigned to a user can be viewed in a list and in a calendar format. Each task comes with a priority level, high priority - red, medium priority - yellow, low priority - green. Blue tasks are already
  complete. Completed tasks are viewable from a previous tasks list. Tasks are assigned a xp (experience points) value when they are created. Users are credited this xp after completion of a task. Xp gained through
  completion of tasks counts towards a user's level. Leveling up unlocks avatars for the user's profile.
